# 飞行多目标跟踪演示平台

基于[Vue.js](https://vuejs.org/)、[Openlayers](https://openlayers.org/)、[Ant Design Vue](https://vue.ant.design)的飞行多目标跟踪演示平台，使用[Vue CLI](https://cli.vuejs.org/)构建。
